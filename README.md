#Dino Google
![](https://i.ibb.co/b7NtJJf/image.png)

El objetivo de este proyecto es recrear el popular juego del dinosaurio de Google que aparece cuando no se tiene acceso a Internet, utilizando tecnologías web como JavaScript, HTML y CSS. El juego del dinosaurio es un juego simple pero adictivo en el que el jugador controla a un dinosaurio que corre automáticamente y debe saltar para evitar obstáculos como cactus y acumular la mayor cantidad de puntos posible.

El proyecto se desarrollará utilizando programación en JavaScript para manejar la lógica del juego, HTML para crear la estructura del juego y los elementos interactivos, y CSS para aplicar estilos visuales y darle una apariencia similar a la del juego original de Google.
